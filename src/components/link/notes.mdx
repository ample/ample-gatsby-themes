# Link

The `<Link />` component determines whether the route passed to it is an internal or external link, and wraps the component appropriately.

If an internal link, the component renders [Gatsby's `<Link />` component](https://www.gatsbyjs.org/docs/gatsby-link/), otherwise it renders an anchor tag.

<Playground>
  <Link to="/components/link">I'm an internal link</Link>
  <br />
  <Link to="https://ample.co">
    <em>And I'm external</em>
  </Link>
</Playground>
